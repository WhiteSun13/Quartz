## § 18. Отрицательные формы глаголов и имён

1. Отрицание при спряжении глаголов в настоящем общем времени выражается последовательным присоединением к основам глаголов аффиксов **-ma / -me**, вставного аффикса **-y**, личных окончаний. Спряжение рассмотрим на примере глаголов **oqumaq** — читать, **beklemek** — ждать.

| Число   | Единственное  |                                                                 | Множественное               |                                                               |
| :------ | :------------ | :-------------------------------------------------------------- | :-------------------------- | :------------------------------------------------------------ |
|         | **Аффиксы**   | **Примеры**                                                     | **Аффиксы**                 | **Примеры**                                                   |
| **I**   | **-ma-y-ım**  | **oqumayım** – я не читаю;<br>**beklemeyim** – я не жду         | **-ma-y-mız**               | **oqumaymız** – мы не читаем;<br>**beklemeymiz** – мы не ждём |
|         | **-me-y-im**  |                                                                 | **-me-y-miz**               |                                                               |
| **II**  | **-ma-y-sıñ** | **oqumaysıñ** – ты не читаешь;<br>**beklemeysiñ** – ты не ждёшь | **-ma-y-**<br>**-sı(ñı)z**, | **oqumays(ıñ)ız** – вы не читаете;                            |
|         | **-me-y-sıñ** |                                                                 | **-me-y-**<br>**-si(ñı)z**  | **beklemeys(iñ)iz** – вы не ждёте                             |
| **III** | **-ma-y,**    | **oqumay** – он не читает;                                      | **-ma-y(-lar)**             | **oqumay(lar)** – они не читают;                              |
|         | **-me-y**     | **beklemey** – он не ждёт                                       | **-me-y(-ler)**             | **beklemey(ler)** – они не ждут                               |

2. Отрицание при спряжении глаголов в простом прошедшем категорическом времени выражается последовательным присоединением к основам глаголов аффиксов **-ma / -me**, временных аффиксов **-dı/-di**, личных окончаний. Спряжение рассмотрим на примере глаголов **içmek** — пить; курить, **yırlamaq** — петь.

| Число | Лицо | Единственное | | Множественное | |
| :--- | :--- | :--- | :--- | :--- | :--- |
| | | **Аффиксы** | **Примеры** | **Аффиксы** | **Примеры** |
| **I** | | **-ma-dı-m,** | **yırlamadım** – я не пел; | **-ma-dıq,** | **yırlamadıq** – мы не пели; |
| | | **-me-di-m** | **içmedim** – я не пил | **-me-dik** | **içmedik** – мы не пили |
| **II** | | **-ma-dı-ñ,** | **yırlamadıñ** – ты не пел; | **-ma-dı-ñız,** | **yırlamadıñız** – вы не пели; |
| | | **-me-di-ñ** | **içmediñ** – ты не пил | **-me-di-ñiz** | **içmediñiz** – вы не пили |
| **III** | | **-ma-dı,** | **yırlamadı** – он не пел; | **-ma-dı**<br>**(-lar),** | **yırlamadı(lar)** – они не пели; |
| | | **-me-di** | **içmedi** – он не пил | **-me-di**<br>**(-ler)** | **içmedi(ler)** – они не пили |

3. Отрицание **degil** — «не» становится после (а не до, как в русском языке!) модальных глаголов (т.е. не имеющих неопределённой формы на **-maq / -mek**; см. тж. урок 6), а также после отрицаемых имён существительных, прилагательных, наречий. *Примеры:* **quş degil** – не птица; **yaramay degil** – неплохой, неплохо; **kerek degil** – не надо; **mümkün degil** – нельзя.

### Til materialı (языковой материал)

#### I. Deñiz yalı sında
Raatlıq künü ava yaramay degil edi, yağmur yağmadı. Bugün Abdulla işke barmay, saba erte kezmege çıqtı. O çayırğa degil, deñiz yalısına ketti. Deñiz tınç edi. Birden deñizden salqın yel esti ve dalğalar yalığa urdı. Suvğa kirmek mümkün degil. Abdulla deñiz yalısında eki saat raatlandı, soñ evge qayttı. Abdulla evde çoq oturmadı ve dostlarına çıqıp ketti.

#### II. Tapmacalar
1) Tili bar – insan degil, tikişi bar – kölmek degil, yaprağı bar – terek degil (*kitap*).
2) Quş degil – uça, adamğa tiyse – yıqa (*qurşun*).
3) Üstü qara – kömür degil, içi beyaz – penir degil, quyruğı uzun – sıçan degil (*qara turup*).
