## Кто такой Python-разработчик

**Python** считается языком для backendразработки. Однако при желании его можно применять и в разработке фронтенда.

**Python-разработчик** — это IT-специалист, который создает сайты, игры, онлайнсервисы, ботов для соцсетей и другое ПО   на языке программирования Python.

## Задачи Python-разработчика зависят от задач компании

***Технические задачи*** направлены на повышение эффективности работы сервисов.

***Бизнес-задачи*** направлены на реализацию нового функционала продукта.

> Результаты выполнения технических задач часто незаметны для команды и пользователей. Результаты бизнесзадач видно сразу

## Для того, чтобы выполнить бизнес-задачу, разработчик

- разрабатывает архитектурное решение;
- пишет код;
- интегрируется в сторонние сервисы (если требуется).

## К техническим задачам относится рефакторинг кода

Примеры рефакторинга (улучшения) кода:
- обновить технический стек (заменить устаревшие инструменты на более   современные);
- оптимизировать код с технической точки  зрения;
- улучшить читаемость кода

## В каких сферах может работать Python-разработчик

**Сферы, в которых применяют Python:**
- веб-разработка;
- мобильная разработка;
- разработка ΠΟ;
- разработка игр;
- машинное обучение;
- DevOps;
- Data Science;
- Data Engineering.

## 3 причины выбрать Python первым языком

1. **Популярность:** у Python огромное комьюнити разработчиков и достаточно много выпущенных материалов для развития своих навыков. Этот язык умрет не скоро.
2. **Возможности:** Python подходит для решения почти всех современных задач, применим во многих сферах разработки.
3. . **Дружелюбный синтаксис:** некоторые детали в Python опущены или упрощены.

# Как выглядит процесс реали- зации проекта?
## За 2 недели разработчики   выполняют следующие действия

- участвуют в оценке реализуемости и сложности крупных бизнес-задач;
- участвуют в формулировании задач разработки;
- участвуют в выборе приоритетных задач на следующий спринт;
- выполняет задачи разработки;
- тестирует код;
- проводит код-ревью;
- участвует в демо, рестроспективах, других встречах.
