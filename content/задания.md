CV = $BCWP - ACWP$
CVP% = $CV / BCWP) * 100
SV = $BCWP - BCWS$
SVP  = (SV / BCWS) * 100

**Расчеты по строкам:**
 
**Работа 1:**
*   $CV = 50000 - 52000 = -2000$
*   $CVP\% = (-2000 / 50000) * 100 = -4\%$
*   $SV = 50000 - 55000 = -5000$
*   $SVP\% = (-5000 / 55000) * 100 \approx -9.1\%$

**Работа 2:**
*   $CV = 42000 - 43500 = -1500$
*   $CVP\% = (-1500 / 42000) * 100 \approx -3.6\%$
*   $SV = 42000 - 42000 = 0$ (Идем точно по графику)
*   $SVP\% = 0\%$

**Работа 3:**
*   $CV = 25000 - 27000 = -2000$
*   $CVP\% = (-2000 / 25000) * 100 = -8\%$
*   $SV = 25000 - 38000 = -13000$ (Сильное отставание)
*   $SVP\% = (-13000 / 38000) * 100 \approx -34.2\%$

**Работа 4:**
*   $CV = 5000 - 3000 = +2000$ (Экономия бюджета!)
*   $CVP\% = (2000 / 5000) * 100 = +40\%$
*   $SV = 5000 - 15000 = -10000$ (Сильное отставание)
*   $SVP\% = (-10000 / 15000) * 100 \approx -66.7\%$

**Строка "Всего":**
Сначала суммируем исходные данные:
*   BCWS Общ = $55000+42000+38000+15000 = 150\,000$
*   BCWP Общ = $50000+42000+25000+5000 = 122\,000$
*   ACWP Общ = $52000+43500+27000+3000 = 125\,500$

Теперь считаем отклонения для всего проекта:
*   $CV \text{ Общ} = 122000 - 125500 = -3500$
*   $CVP\% \text{ Общ} = (-3500 / 122000) * 100 \approx -2.9\%$
*   $SV \text{ Общ} = 122000 - 150000 = -28000$
*   $SVP\% \text{ Общ} = (-28000 / 150000) * 100 \approx -18.7\%$
