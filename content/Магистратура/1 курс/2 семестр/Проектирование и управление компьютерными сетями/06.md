### 💡 Лабораторные технологии: VLAN, VRRP, Q-in-Q

| Технология | Описание |
| :--- | :--- |
| **VLAN** | Разделение сети на уровне коммутатора для изоляции трафика. |
| **VRRP / HSRP** | Протоколы резервирования шлюза. Два роутера используют один виртуальный IP, при падении одного другой перехватывает трафик. |
| **Q-in-Q** | Упаковка одного VLAN внутрь другого VLAN. Полезно для провайдеров, чтобы прогонять клиентские VLAN через свою сеть без конфликта ID. |
| **MPLS (L3 VPN)** | Маршрутизация по меткам (`labels`), ускоряющая передачу данных и позволяющая строить туннели между узлами (часто у провайдеров связи). |
| **Router on a Stick** | Маршрутизация между VLAN-ами через один физический интерфейс роутера с использованием подинтерфейсов (виртуальных интерфейсов). |
