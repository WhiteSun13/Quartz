Конечно! **Метод простых итераций** — это фундаментальный и один из самых интуитивно понятных итерационных методов. Он применяется для решения широкого круга задач, включая решение нелинейных уравнений и систем линейных алгебраических уравнений (СЛАУ).

Давайте рассмотрим его применение в обеих областях.

### **Часть I: Метод простых итераций для решения уравнений `f(x) = 0`**

#### **1. Основная идея**

Идея состоит в том, чтобы преобразовать исходное уравнение `f(x) = 0` в эквивалентную ему форму:

`x = φ(x)`

Такое представление называется **итерационной формой**, а функция `φ(x)` — **итерационной функцией**.

После того как мы получили такую форму, мы можем построить итерационный процесс:
1.  Выбрать начальное приближение `x₀`.
2.  Применять итерационную формулу для получения последовательности приближений:
    `x₁ = φ(x₀)`
    `x₂ = φ(x₁)`
    ...
    `x_{k+1} = φ(x_k)`

Если процесс сходится, то последовательность `x₀, x₁, x₂, ...` будет стремиться к некоторому значению `x*`, которое и является решением уравнения, так как в пределе `x* = φ(x*)`.

#### **2. Пример**

Рассмотрим уравнение: `x³ + x - 1 = 0`.
Как его привести к виду `x = φ(x)`? Есть много способов:

1.  **Способ 1:** `x = 1 - x³`. Здесь `φ₁(x) = 1 - x³`.
2.  **Способ 2:** `x(x² + 1) = 1  =>  x = 1 / (x² + 1)`. Здесь `φ₂(x) = 1 / (x² + 1)`.
3.  **Способ 3:** `x³ = 1 - x  =>  x = ³√(1 - x)`. Здесь `φ₃(x) = (1 - x)^(1/3)`.

От выбора функции `φ(x)` критически зависит, будет ли метод сходиться.

#### **3. Условие сходимости**

Это самый важный аспект метода. Процесс `x_{k+1} = φ(x_k)` сходится к корню `x*` на отрезке `[a, b]`, если выполнены два условия:

1.  **Функция `φ(x)` отображает отрезок `[a, b]` в себя.** То есть, для любого `x` из `[a, b]`, значение `φ(x)` также должно лежать в `[a, b]`. Это удерживает итерации внутри отрезка.
2.  **Функция `φ(x)` является сжимающим отображением на `[a, b]`**. Это ключевое условие. Математически оно означает, что существует константа `q < 1` такая, что для любых `x₁, x₂` из `[a, b]` выполняется `|φ(x₁) - φ(x₂)| ≤ q * |x₁ - x₂|`.

На практике это сложное условие заменяют более простым **достаточным условием сходимости**:

> **Если на отрезке `[a, b]`, содержащем корень, производная `|φ'(x)| ≤ q < 1` для всех `x`, то метод простых итераций сходится.**

**Интуиция:** Если модуль производной меньше 1, то функция "сжимает" расстояние между точками. Каждая следующая итерация будет ближе к корню, чем предыдущая. Если `|φ'(x)| > 1`, то итерации будут "разбегаться" от корня.

**Вернемся к нашему примеру.** Корень уравнения `x³ + x - 1 = 0` находится примерно в `x ≈ 0.68`. Проверим производные наших `φ(x)` в окрестности этого корня:
1.  `φ₁'(x) = -3x²`. `|φ₁'(0.68)| = |-3 * (0.68)²| ≈ 1.38 > 1`. **Процесс разойдется.**
2.  `φ₂'(x) = -2x / (x² + 1)²`. `|φ₂'(0.68)| ≈ |-1.36 / (1.46)²| ≈ 0.64 < 1`. **Процесс сойдется!**
3.  `φ₃'(x) = -1 / (3 * (1-x)^(2/3))`. `|φ₃'(0.68)| ≈ |-1 / (3 * (0.32)^(2/3))| ≈ 0.7 < 1`. **Процесс тоже сойдется.**

#### **4. Геометрическая интерпретация**

Процесс итераций можно представить графически. Мы ищем точку пересечения графиков `y = x` и `y = φ(x)`.
1.  Начинаем с `x₀`.
2.  Находим `y₀ = φ(x₀)`.
3.  Чтобы получить `x₁`, мы должны "перенести" значение `y₀` на ось X. Это делается движением по горизонтали до прямой `y=x`. Точка на оси X под ней и будет `x₁ = y₀`.
4.  Повторяем процесс.

### **Часть II: Метод простых итераций для решения СЛАУ `Ax = b` (Метод Якоби)**

Для систем линейных уравнений **метод простой итерации** известен под названием **метод Якоби**.

#### **1. Идея и вывод формулы**

1.  Представим матрицу `A` в виде `A = L + D + U`, где `L` — строго нижняя треугольная часть, `D` — диагональ, `U` — строго верхняя треугольная часть.
2.  Исходное уравнение `Ax = b` перепишем как `(L + D + U)x = b`.
3.  Выразим `Dx`: `Dx = b - (L + U)x`.
4.  Умножим на `D⁻¹` (это просто матрица с `1/a_ii` на диагонали):
    `x = -D⁻¹(L + U)x + D⁻¹b`

Мы получили канонический вид `x = Bx + c`, где:
*   `B = -D⁻¹(L + U)`
*   `c = D⁻¹b`

#### **2. Итерационная формула**

Итерационный процесс строится на основе этой формы:

`x⁽ᵏ⁺¹⁾ = -D⁻¹(L + U)x⁽ᵏ⁾ + D⁻¹b`

В поэлементной форме это выглядит так (именно эта форма используется в программах):

`x_i⁽ᵏ⁺¹⁾ = (1/a_ii) * (b_i - Σ(j≠i) a_ij * x_j⁽ᵏ⁾)`

**Словесное описание:** Чтобы найти `i`-ю компоненту нового приближения, мы берем `i`-е уравнение, переносим все члены, кроме `a_ii * x_i`, в правую часть и делим на `a_ii`. При этом в правой части используются значения всех переменных из **предыдущего** приближения `x⁽ᵏ⁾`.

#### **3. Условие сходимости**

Как и в общем случае, метод сходится, если норма матрицы перехода `B = -D⁻¹(L + U)` меньше единицы.

На практике используют более простое **достаточное условие**:

> **Метод Якоби сходится, если матрица `A` имеет строгое диагональное преобладание.**
>
> `|a_ii| > Σ(j≠i) |a_ij|` для `i = 1, ..., n`
> (Модуль диагонального элемента в каждой строке строго больше суммы модулей остальных элементов этой строки).

#### **4. Преимущества и недостатки**

*   **Преимущества:**
    *   Простота реализации.
    *   **Идеален для распараллеливания:** поскольку все компоненты `x_i⁽ᵏ⁺¹⁾` вычисляются независимо друг от друга на основе старого вектора `x⁽ᵏ⁾`, их можно рассчитывать на разных процессорах или ядрах одновременно.
*   **Недостатки:**
    *   Сходимость относительно медленная по сравнению с более продвинутыми методами (например, методом Гаусса-Зейделя или сопряженных градиентов).
    *   Требование диагонального преобладания довольно жесткое.

**Вывод:** Метод простых итераций (и его частный случай — метод Якоби) является базовым и важным для понимания итерационных процессов. Хотя существуют более быстрые методы, его простота и возможность легкого распараллеливания делают его актуальным и сегодня, особенно в контексте высокопроизводительных вычислений.
