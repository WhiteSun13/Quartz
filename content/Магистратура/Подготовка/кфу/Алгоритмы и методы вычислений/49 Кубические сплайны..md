Отлично! **Кубические сплайны** — это, пожалуй, самый важный, мощный и широко используемый метод интерполяции в современных вычислениях. Давайте разберем его во всех деталях.

### **Кубические Сплайны: Полное Руководство**

#### **1. Мотивация: Почему не полином Лагранжа?**

Как мы уже знаем, попытка провести через большое количество точек $(n+1)$ один-единственный многочлен степени $n$ приводит к **феномену Рунге**: между узлами возникают сильные нефизичные колебания (осцилляции), особенно по краям интервала. Это делает полиномиальную интерполяцию высокой степени практически бесполезной для аппроксимации данных.

**Идея сплайнов:** Вместо одного сложного многочлена, давайте использовать много простых! Мы разобьем весь интервал $[x_0, x_n]$ на $n$ маленьких отрезков $[x_i, x_{i+1}]$ и на каждом из них построим свой собственный многочлен **низкой степени**. Затем мы "склеим" эти многочлены в узлах так, чтобы итоговая кривая была не только непрерывной, но и гладкой.

*   **Линейный сплайн (ломаная):** На каждом отрезке — полином 1-й степени. Кривая непрерывна, но не гладкая (имеет изломы в узлах).
*   **Квадратичный сплайн:** На каждом отрезке — полином 2-й степени. Можно добиться гладкости первого порядка (непрерывность касательной), но возникают сложности с заданием граничных условий. Используется реже.
*   **Кубический сплайн:** На каждом отрезке — полином 3-й степени. Это "золотая середина", позволяющая добиться идеального баланса между гибкостью, гладкостью и вычислительной сложностью.

#### **2. Математическое Определение Кубического Сплайна**

Пусть дан набор из $n+1$ узла $(x_0, y_0), (x_1, y_1), ..., (x_n, y_n)$.
Функция $S(x)$ называется **кубическим сплайном**, если она удовлетворяет следующим условиям:

1.  **Кусочно-кубическая:** На каждом подынтервале $[x_i, x_{i+1}]$ (для $i=0, ..., n-1$) функция $S(x)$ является многочленом третьей степени. Обозначим его как $S_i(x)$.
    $S_i(x) = a_i + b_i(x - x_i) + c_i(x - x_i)^2 + d_i(x - x_i)^3$

2.  **Условия интерполяции:** Сплайн должен проходить через все заданные точки.
    *   $S_i(x_i) = y_i$ для $i = 0, ..., n-1$
    *   $S_i(x_{i+1}) = y_{i+1}$ для $i = 0, ..., n-1$

3.  **Условия гладкости (непрерывности производных):** В каждом *внутреннем* узле $x_i$ (для $i=1, ..., n-1$) производные соседних полиномов должны совпадать.
    *   **Непрерывность первой производной (гладкость наклона):** $S'_{i-1}(x_i) = S'_i(x_i)$
    *   **Непрерывность второй производной (гладкость кривизны):** $S''_{i-1}(x_i) = S''_i(x_i)$

**Физическая аналогия:** Представьте себе гибкую металлическую линейку (лекало), которую инженеры изгибают так, чтобы она прошла через гвоздики, вбитые в точках $(x_i, y_i)$. Форма, которую примет такая линейка, очень близка к кубическому сплайну. Условие непрерывности второй производной как раз и минимизирует энергию изгиба.

#### **3. Алгоритм Построения (Вывод Системы Уравнений)**

Нам нужно найти $4n$ коэффициентов ($a_i, b_i, c_i, d_i$ для каждого из $n$ интервалов). Давайте посчитаем, сколько у нас уравнений из условий выше:

*   $S_i(x_i) = y_i$ дает $n$ уравнений.
*   $S_i(x_{i+1}) = y_{i+1}$ дает $n$ уравнений.
*   $S'_{i-1}(x_i) = S'_i(x_i)$ дает $n-1$ уравнение.
*   $S''_{i-1}(x_i) = S''_i(x_i)$ дает $n-1$ уравнение.

**Итого:** $n + n + (n-1) + (n-1) = 4n - 2$ уравнений.
У нас $4n$ неизвестных и $4n-2$ уравнения. Нам не хватает **двух** уравнений!

Эти два недостающих уравнения получаются из **граничных условий**, которые определяют поведение сплайна на концах интервала (в точках $x_0$ и $x_n$).

##### **3.1. Вывод уравнений в явном виде**

Давайте выведем систему. Обозначим $h_i = x_{i+1} - x_i$.

1.  Из условия $S_i(x_i) = y_i$ сразу следует:
    $a_i = y_i$

2.  Запишем сам полином и его производные:
    *   $S_i(x) = y_i + b_i(x-x_i) + c_i(x-x_i)^2 + d_i(x-x_i)^3$
    *   $S'_i(x) = b_i + 2c_i(x-x_i) + 3d_i(x-x_i)^2$
    *   $S''_i(x) = 2c_i + 6d_i(x-x_i)$

3.  Из условия $S_i(x_{i+1}) = y_{i+1}$:
    $y_{i+1} = y_i + b_i h_i + c_i h_i^2 + d_i h_i^3$ (Уравнение А)

4.  Из условия непрерывности производных:
    *   $S'_{i-1}(x_i) = S'_i(x_i) \implies b_{i-1} + 2c_{i-1}h_{i-1} + 3d_{i-1}h_{i-1}^2 = b_i$ (Уравнение B)
    *   $S''_{i-1}(x_i) = S''_i(x_i) \implies 2c_{i-1} + 6d_{i-1}h_{i-1} = 2c_i \implies c_i = c_{i-1} + 3d_{i-1}h_{i-1}$ (Уравнение C)

Решать эту систему напрямую для $b, c, d$ неудобно. Существует более элегантный подход, где основной неизвестной является вторая производная.

##### **3.2. Стандартный метод решения через вторые производные**

Обозначим $m_i = S''(x_i)$ — значения вторых производных в узлах. Так как $S_i''(x)$ — это линейная функция, на отрезке $[x_i, x_{i+1}]$ ее можно записать как:
$S_i''(x) = m_i \frac{x_{i+1} - x}{h_i} + m_{i+1} \frac{x - x_i}{h_i}$

Проинтегрировав это выражение дважды и использовав условия интерполяции, можно (после довольно громоздких выкладок) получить связь между $m_i$:
$h_{i-1}m_{i-1} + 2(h_{i-1} + h_i)m_i + h_i m_{i+1} = 6 \left( \frac{y_{i+1}-y_i}{h_i} - \frac{y_i - y_{i-1}}{h_{i-1}} \right)$

Это система из $n-1$ линейных уравнений для $n+1$ неизвестных $m_0, m_1, ..., m_n$. Снова не хватает двух уравнений.

#### **4. Граничные Условия**

Вот самые популярные способы задать два недостающих условия:

1.  **Естественный сплайн (Natural Spline):** Самый распространенный тип. Мы предполагаем, что на концах "гибкая линейка" не испытывает изгибающего момента.
    $m_0 = S''(x_0) = 0$
    $m_n = S''(x_n) = 0$
    Это дает два недостающих уравнения. Система становится замкнутой и имеет единственное решение.

2.  **Закрепленный сплайн (Clamped Spline):** Используется, если мы знаем точные значения первой производной на концах, например, $f'(x_0) = \alpha$ и $f'(x_n) = \beta$.
    $S'(x_0) = \alpha$
    $S'(x_n) = \beta$
    Эти условия также позволяют получить два уравнения для $m_i$ и замкнуть систему. Этот сплайн обычно дает более точное приближение, если производные известны.

3.  **"Не-узел" (Not-a-Knot):** Очень умный способ, используемый по умолчанию, например, в MATLAB. Мы требуем, чтобы третья производная также была непрерывной в первом и последнем внутренних узлах ($x_1$ и $x_{n-1}$).
    $S'''_0(x_1) = S'''_1(x_1)$
    $S'''_{n-2}(x_{n-1}) = S'''_{n-1}(x_{n-1})$
    Это фактически означает, что первый и второй кубические полиномы являются одним и тем же полиномом, как и два последних.

#### **5. Решение Системы и Построение Сплайна**

Какое бы граничное условие мы ни выбрали, мы получаем **систему линейных алгебраических уравнений (СЛАУ)** для нахождения $m_1, ..., m_{n-1}$ (или $m_0, ..., m_n$).

Ключевое свойство этой СЛАУ: ее матрица является **трехдиагональной**. Это означает, что ненулевые элементы есть только на главной диагонали и двух соседних с ней.

**Пример матрицы для естественного сплайна (равномерная сетка, $h_i=h$):**
$\begin{pmatrix}
4 & 1 & 0 & \cdots & 0 \\
1 & 4 & 1 & \cdots & 0 \\
0 & 1 & 4 & \cdots & 0 \\
\vdots & \vdots & \vdots & \ddots & \vdots \\
0 & 0 & 0 & \cdots & 4
\end{pmatrix}
\begin{pmatrix}
m_1 \\ m_2 \\ m_3 \\ \vdots \\ m_{n-1}
\end{pmatrix}
=
\frac{6}{h^2}
\begin{pmatrix}
y_2 - 2y_1 + y_0 \\
y_3 - 2y_2 + y_1 \\
y_4 - 2y_3 + y_2 \\
\vdots \\
y_n - 2y_{n-1} + y_{n-2}
\end{pmatrix}$

**Преимущество трехдиагональной матрицы:** Системы с такими матрицами решаются не стандартным методом Гаусса за $O(n^3)$, а гораздо более эффективным **методом прогонки (алгоритм Томаса)** всего за $O(n)$ операций. Это делает построение сплайнов очень быстрым даже для тысяч точек.

**Итоговый алгоритм:**

1.  **Выбор граничных условий:** Определяемся, какой тип сплайна нам нужен (например, естественный).
2.  **Формирование СЛАУ:** Для каждого внутреннего узла $i=1, ..., n-1$ вычисляем правую часть уравнения и записываем коэффициенты трехдиагональной матрицы.
3.  **Решение СЛАУ:** Решаем систему методом прогонки и находим все значения вторых производных $m_0, m_1, ..., m_n$.
4.  **Нахождение коэффициентов:** Зная $m_i$, легко найти остальные коэффициенты для каждого полинома $S_i(x)$:
    *   $a_i = y_i$
    *   $d_i = \frac{m_{i+1} - m_i}{6h_i}$
    *   $c_i = \frac{m_i}{2}$
    *   $b_i = \frac{y_{i+1} - y_i}{h_i} - \frac{h_i(2m_i + m_{i+1})}{6}$
5.  **Интерполяция:** Чтобы найти значение в произвольной точке $x$, мы сначала определяем, в какой интервал $[x_i, x_{i+1}]$ она попадает, а затем вычисляем значение по формуле для $S_i(x)$.

#### **6. Преимущества и Недостатки Кубических Сплайнов**

| Преимущества | Недостатки |
| :--- | :--- |
| ✅ **Отсутствие феномена Рунге:** Сплайны не осциллируют дико, даже при очень большом числе точек. | ❌ **Более сложная реализация:** Требуется решить СЛАУ, что сложнее, чем просто подставить числа в формулу Лагранжа. |
| ✅ **Высокая гладкость ($C^2$):** Итоговая кривая не только непрерывна, но и имеет непрерывные первую и вторую производные. Это дает очень "естественный" и приятный вид. | ❌ **Глобальная зависимость:** Хотя изменение одной точки $(x_k, y_k)$ не меняет форму сплайна далеко от нее, формально для пересчета нужно решать всю СЛАУ заново. Значение $m_i$ зависит от всех $y_j$. |
| ✅ **Вычислительная эффективность:** Несмотря на кажущуюся сложность, построение сплайна (решение СЛАУ) и его вычисление очень быстрые — $O(n)$. | ❌ **Требуются граничные условия:** Необходимо делать предположение о поведении функции на концах, что может повлиять на результат. |
| ✅ **Хорошие аппроксимационные свойства:** Погрешность интерполяции кубическим сплайном имеет порядок $O(h^4)$, где $h$ — максимальный шаг сетки. Это очень высокая точность. | |

#### **7. Практическое Применение**

Кубические сплайны — рабочая лошадка в огромном количестве областей:
*   **Компьютерная графика:** Для построения гладких кривых, траекторий движения камеры, анимации персонажей. Шрифты (TrueType, PostScript) используют кубические или квадратичные кривые Безье, которые тесно связаны со сплайнами.
*   **Инженерное проектирование (CAD/CAM):** Для описания контуров деталей, корпусов автомобилей, самолетов.
*   **Робототехника:** Для планирования плавных траекторий движения манипуляторов.
*   **Анализ данных и визуализация:** Для построения гладких графиков по дискретным данным измерений.
*   **Численные методы:** Для высокоточного вычисления интегралов и производных от таблично-заданных функций.

**Вывод:** Кубические сплайны — это мощный, надежный и эффективный инструмент для гладкой интерполяции данных. Они успешно преодолевают недостатки полиномиальной интерполяции и являются де-факто стандартом для большинства практических задач, где требуется восстановить гладкую зависимость по дискретным точкам.
