---

### **Интерполяционный Метод Лагранжа**

**Задача:**
У нас есть набор из $n+1$ точки $(x_0, y_0), (x_1, y_1), ..., (x_n, y_n)$, где все узлы $x_i$ различны. Мы хотим найти **единственный** многочлен $P_n(x)$ степени не выше $n$, который проходит в точности через все эти точки, то есть удовлетворяет условиям:
$P_n(x_i) = y_i$ для всех $i = 0, 1, ..., n$.

**Основная идея метода Лагранжа:**
Идея гениальна в своей простоте. Вместо того чтобы искать коэффициенты многочлена $P_n(x) = a_n x^n + ... + a_1 x + a_0$ путем решения сложной системы линейных уравнений, Жозеф Луи Лагранж предложил построить искомый многочлен как **взвешенную сумму** специальных, более простых "базисных" многочленов.

Каждый базисный многочлен $L_i(x)$ обладает уникальным свойством:
*   Он равен **1** в "своём" узле $x_i$.
*   Он равен **0** во всех *остальных* узлах $x_j$ (где $j \neq i$).

То есть, $L_i(x_j) = \begin{cases} 1, & \text{если } j=i \\ 0, & \text{если } j \neq i \end{cases}$

Представьте эти базисные многочлены как "переключатели". $L_i(x)$ "включает" значение $y_i$ в точке $x_i$ и "выключает" все остальные.

Если у нас есть такие базисные многочлены, то итоговый полином $P_n(x)$ можно очень легко сконструировать:
$P_n(x) = y_0 \cdot L_0(x) + y_1 \cdot L_1(x) + ... + y_n \cdot L_n(x) = \sum_{i=0}^{n} y_i L_i(x)$

**Почему это работает?** Проверим, проходит ли этот полином через узел, например, $x_k$:
$P_n(x_k) = \sum_{i=0}^{n} y_i L_i(x_k) = y_0 L_0(x_k) + y_1 L_1(x_k) + ... + y_k L_k(x_k) + ... + y_n L_n(x_k)$

По определению базисных полиномов, все $L_i(x_k)$ равны 0, кроме $L_k(x_k)$, который равен 1.
$P_n(x_k) = y_0 \cdot 0 + y_1 \cdot 0 + ... + y_k \cdot 1 + ... + y_n \cdot 0 = y_k$
Условие $P_n(x_k) = y_k$ выполняется для любого узла $k$. Задача решена!

### **Математическая Формула**

Теперь главный вопрос: как построить сами базисные многочлены $L_i(x)$?

1.  **Как обеспечить нули?** Чтобы многочлен был равен нулю в точках $x_0, x_1, ..., x_{i-1}, x_{i+1}, ..., x_n$, он должен содержать множители $(x-x_0), (x-x_1), ...$ и т.д.
    То есть, он должен быть пропорционален произведению: $(x-x_0)(x-x_1)...(x-x_{i-1})(x-x_{i+1})...(x-x_n)$.
    В компактной форме: $\prod_{j=0, j \neq i}^{n} (x - x_j)$.

2.  **Как обеспечить единицу?** Теперь нам нужно, чтобы в точке $x_i$ этот многочлен был равен 1. Подставим $x_i$ в наше произведение:
    $\prod_{j=0, j \neq i}^{n} (x_i - x_j)$
    Это просто число (константа). Чтобы получить 1, нам нужно разделить исходное произведение на эту константу.

Соединяя эти два шага, мы получаем формулу для **базисного полинома Лагранжа $L_i(x)$**:

$L_i(x) = \frac{(x-x_0)(x-x_1)...(x-x_{i-1})(x-x_{i+1})...(x-x_n)}{(x_i-x_0)(x_i-x_1)...(x_i-x_{i-1})(x_i-x_{i+1})...(x_i-x_n)} = \prod_{j=0, j \neq i}^{n} \frac{x - x_j}{x_i - x_j}$

Итоговая **формула интерполяционного многочлена Лагранжа**:

$P_n(x) = \sum_{i=0}^{n} y_i \left( \prod_{j=0, j \neq i}^{n} \frac{x - x_j}{x_i - x_j} \right)$

### **Пошаговый Пример**

Давайте построим интерполяционный многочлен для трех точек:
**(1, 2), (2, 3), (4, 1)**

Здесь $n=2$, нам нужен многочлен степени не выше 2.
*   $(x_0, y_0) = (1, 2)$
*   $(x_1, y_1) = (2, 3)$
*   $(x_2, y_2) = (4, 1)$

**Шаг 1: Строим базисные многочлены**

*   **Для $i=0$ (точка (1, 2)):**
    $L_0(x) = \frac{(x - x_1)(x - x_2)}{(x_0 - x_1)(x_0 - x_2)} = \frac{(x - 2)(x - 4)}{(1 - 2)(1 - 4)} = \frac{x^2 - 6x + 8}{(-1)(-3)} = \frac{1}{3}(x^2 - 6x + 8)$

*   **Для $i=1$ (точка (2, 3)):**
    $L_1(x) = \frac{(x - x_0)(x - x_2)}{(x_1 - x_0)(x_1 - x_2)} = \frac{(x - 1)(x - 4)}{(2 - 1)(2 - 4)} = \frac{x^2 - 5x + 4}{(1)(-2)} = -\frac{1}{2}(x^2 - 5x + 4)$

*   **Для $i=2$ (точка (4, 1)):**
    $L_2(x) = \frac{(x - x_0)(x - x_1)}{(x_2 - x_0)(x_2 - x_1)} = \frac{(x - 1)(x - 2)}{(4 - 1)(4 - 2)} = \frac{x^2 - 3x + 2}{(3)(2)} = \frac{1}{6}(x^2 - 3x + 2)$

**Шаг 2: Собираем итоговый многочлен**
$P_2(x) = y_0 L_0(x) + y_1 L_1(x) + y_2 L_2(x)$
$P_2(x) = 2 \cdot \left[\frac{1}{3}(x^2 - 6x + 8)\right] + 3 \cdot \left[-\frac{1}{2}(x^2 - 5x + 4)\right] + 1 \cdot \left[\frac{1}{6}(x^2 - 3x + 2)\right]$

**Шаг 3: Упрощаем выражение (раскрываем скобки и приводим подобные слагаемые)**
$P_2(x) = \left(\frac{2}{3} - \frac{3}{2} + \frac{1}{6}\right)x^2 + \left(-\frac{12}{3} + \frac{15}{2} - \frac{3}{6}\right)x + \left(\frac{16}{3} - \frac{12}{2} + \frac{2}{6}\right)$

Приводим к общему знаменателю (6):
$P_2(x) = \left(\frac{4 - 9 + 1}{6}\right)x^2 + \left(\frac{-24 + 45 - 3}{6}\right)x + \left(\frac{32 - 36 + 2}{6}\right)$
$P_2(x) = \left(\frac{-4}{6}\right)x^2 + \left(\frac{18}{6}\right)x + \left(\frac{-2}{6}\right)$

$P_2(x) = -\frac{2}{3}x^2 + 3x - \frac{1}{3}$

**Шаг 4: Проверка**
Давайте убедимся, что наш многочлен проходит через заданные точки.
*   $P_2(1) = -\frac{2}{3}(1)^2 + 3(1) - \frac{1}{3} = -\frac{2}{3} + 3 - \frac{1}{3} = -\frac{3}{3} + 3 = -1 + 3 = 2$. **Верно.**
*   $P_2(2) = -\frac{2}{3}(2)^2 + 3(2) - \frac{1}{3} = -\frac{8}{3} + 6 - \frac{1}{3} = -\frac{9}{3} + 6 = -3 + 6 = 3$. **Верно.**
*   $P_2(4) = -\frac{2}{3}(4)^2 + 3(4) - \frac{1}{3} = -\frac{32}{3} + 12 - \frac{1}{3} = -\frac{33}{3} + 12 = -11 + 12 = 1$. **Верно.**

### **Преимущества и Недостатки Метода Лагранжа**

#### **Преимущества:**

1.  **Концептуальная простота:** Формула легко выводится и запоминается, идея с базисными полиномами очень наглядна.
2.  **Прямое вычисление:** Для нахождения значения $P_n(x)$ в конкретной точке $x$ не требуется предварительно находить коэффициенты $a_i$ и решать СЛАУ. Можно просто подставить $x$ в общую формулу.
3.  **Теоретическая значимость:** Формула Лагранжа является конструктивным доказательством теоремы о существовании и единственности интерполяционного многочлена.

#### **Недостатки:**

1.  **Вычислительная неэффективность:** Для вычисления значения $P_n(x)$ в одной точке требуется выполнить порядка $O(n^2)$ арифметических операций (внешняя сумма из $n+1$ слагаемого, каждое из которых требует вычисления произведения из $n$ множителей). Для большого числа узлов это очень медленно.
2.  **Нестабильность при добавлении узлов:** Если к набору точек добавляется новая точка $(x_{n+1}, y_{n+1})$, весь многочлен нужно пересчитывать с нуля. Все базисные полиномы $L_i(x)$ изменятся, так как изменится набор узлов. В этом плане **интерполяционная форма Ньютона** гораздо более гибкая, так как она позволяет просто добавить новое слагаемое к уже существующему многочлену.
3.  **Возможные вычислительные погрешности:** При большом $n$ значения числителей и знаменателей в базисных полиномах могут становиться очень большими или очень маленькими, что может приводить к потере точности при вычислениях с плавающей запятой.

---

### **Вывод и Практическое Применение**

Несмотря на свои вычислительные недостатки, метод Лагранжа является краеугольным камнем в теории интерполяции. Его изучают в первую очередь из-за его **ясности и теоретической элегантности**.

**На практике:**
*   Для **ручных расчетов** с небольшим количеством точек (3-5) метод Лагранжа вполне удобен.
*   В **программных реализациях**, когда важна производительность и гибкость, чаще используют **форму Ньютона** (из-за быстрого вычисления по схеме Горнера и легкости добавления новых узлов) или **кубические сплайны** (чтобы избежать феномена Рунге при большом числе точек).
*   Формула Лагранжа часто используется в **теоретических выкладках** и доказательствах, например, при выводе формул численного интегрирования (формулы Ньютона-Котеса).

Таким образом, многочлен Лагранжа — это фундаментальный инструмент, который прекрасно иллюстрирует саму суть полиномиальной интерполяции, но в реальных вычислительных задачах с большим объемом данных он уступает более эффективным алгоритмам.
