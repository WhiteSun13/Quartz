- Любой код - метод некоторого класса

Классы находятся в пакете.

Объект передаётся по ссылке

```Java
Employee hacker = new Employee("Bob");

```

private - только калссу
package-private - только пакету
protected - класс, пакет, класс-наследник
public - всем
