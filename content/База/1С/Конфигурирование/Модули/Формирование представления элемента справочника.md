# Формирование представления элемента справочника

## Зачем это нужно

По умолчанию платформа показывает в заголовке карточки и в списке
значение реквизита «Наименование». Через модуль менеджера можно
формировать **произвольный заголовок** — например, добавить возраст
сотрудника или статус записи.

## Два обработчика и их параметры

Оба обработчика находятся в **модуле менеджера**.

### ОбработкаПолученияПолейПредставления

Сообщает платформе, **какие реквизиты** нужно загрузить
для формирования заголовка.

| Параметр | Тип | Описание |
|---|---|---|
| `Поля` | Массив строк | Сюда добавляем имена нужных реквизитов |
| `СтандартнаяОбработка` | Булево | `Ложь` — отключает стандартное поведение |

```bsl
Процедура ОбработкаПолученияПолейПредставления(Поля, СтандартнаяОбработка)
    СтандартнаяОбработка = Ложь;
    Поля.Добавить("Наименование");
    Поля.Добавить("ДатаРождения");
КонецПроцедуры
```

> [!note]
> Реквизиты добавляются как **строки** (в двойных кавычках).
> Контекстная подсказка здесь не работает — платформа
> не распознаёт тип параметра `Поля` автоматически.

> [!tip]
> Чтобы не допустить опечатку в имени реквизита — перетащите его
> из дерева конфигурации в модуль, зажав левую кнопку мыши.
> Платформа подставит корректное имя автоматически.

### ОбработкаПолученияПредставления

Формирует **итоговый заголовок**, который увидит пользователь.

| Параметр | Тип | Описание |
|---|---|---|
| `Данные` | Структура | Содержит реквизиты, добавленные через `Поля` |
| `Представление` | Строка | Сюда помещаем готовый заголовок |
| `СтандартнаяОбработка` | Булево | `Ложь` — отключает стандартное поведение |

> [!important]
> Чтобы переопределить стандартный заголовок — в **обоих**
> обработчиках обязательно пишем:
> ```bsl
> СтандартнаяОбработка = Ложь;
> ```

## Полный пример: ФИО + возраст в заголовке

```bsl
Процедура ОбработкаПолученияПолейПредставления(Поля, СтандартнаяОбработка)
    СтандартнаяОбработка = Ложь;
    Поля.Добавить("Наименование");
    Поля.Добавить("ДатаРождения");
КонецПроцедуры

Процедура ОбработкаПолученияПредставления(Данные, Представление, СтандартнаяОбработка)
    СтандартнаяОбработка = Ложь;

    Если Данные.ДатаРождения > '00010101' Тогда
        // Расчёт возраста
        ВозрастВСекундах = ТекущаяДата() - Данные.ДатаРождения;
        Возраст1 = '00010101' + ВозрастВСекундах;
        Возраст2 = Возраст1 - 86400;
        Возраст  = Год(Возраст2) - 1;

        // Склонение слова
        ПоследняяЦифра = Возраст % 10;
        Если ПоследняяЦифра = 0
            Или (ПоследняяЦифра >= 5 И ПоследняяЦифра <= 9)
            Или (Возраст >= 11 И Возраст <= 14) Тогда
            ФормаСлова = "лет";
        ИначеЕсли ПоследняяЦифра = 1 Тогда
            ФормаСлова = "год";
        ИначеЕсли ПоследняяЦифра >= 2 И ПоследняяЦифра <= 4 Тогда
            ФормаСлова = "года";
        КонецЕсли;

        // Итоговый заголовок
        Представление = СтрШаблон("%1 (%2 %3)",
            Данные.Наименование,
            Возраст,
            ФормаСлова);
    Иначе
        // Дата рождения не заполнена — только ФИО
        Представление = Данные.Наименование;
    КонецЕсли;
КонецПроцедуры
```

**Результат в интерфейсе:**
```
Петров П.П. (22 года)
Механизмов А.В. (45 лет)
Сидорова О.К. (21 год)
```
