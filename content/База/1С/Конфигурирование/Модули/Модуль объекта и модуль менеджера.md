# Модуль объекта и модуль менеджера

## Где найти модули

1. Открыть дерево конфигурации → нужный объект (справочник, документ)
2. Двойной клик → вкладка **«Прочие»**
3. Кнопки **«Модуль объекта»** и **«Модуль менеджера»**

*Альтернатива:* правая кнопка мыши по объекту → выбрать нужный модуль.

## Сравнение модулей

| Характеристика | Модуль объекта | Модуль менеджера |
|---|---|---|
| **На что влияет** | На **один конкретный элемент** | На **все элементы** объекта |
| **Когда запускается** | При открытии или записи **конкретной карточки** | При открытии **всего списка** |
| **Типичное применение** | Вычисление реквизитов при записи | Формирование заголовков, представлений |

> [!tip]
> **Простая аналогия:**
> - **Модуль объекта:** открыли карточку Механизмова —
>   алгоритм сработал только для него.
> - **Модуль менеджера:** открыли список «Сотрудники» —
>   алгоритм отработал для всех записей сразу.

## Обработчики событий модуля объекта

| Обработчик | Назначение |
|---|---|
| **Перед записью** | Дозаполнение данных перед сохранением |
| **При записи** | Контроль правильности данных |

> [!important]
> По стандартам 1С:
> - **Перед записью** — для **дозаполнения** (вычисляем и подставляем значения)
> - **При записи** — для **контроля** (проверяем, что данные корректны)

## Обработчики событий модуля менеджера

| Обработчик | Назначение |
|---|---|
| **ОбработкаПолученияПолейПредставления** | Указывает, какие реквизиты нужны для формирования заголовка |
| **ОбработкаПолученияПредставления** | Формирует итоговый заголовок на основе полученных данных |

Подробнее о формировании представления:
см. [Формирование представления элемента справочника](1c-form-representation.md).

## Пример: формирование ФИО при записи (модуль объекта)

```bsl
Процедура ПередЗаписью(Отказ)
    Наименование = СтрШаблон("%1 %2.%3.",
        СокрЛП(Фамилия),
        Лев(ВРег(СокрЛП(Имя)), 1),
        Лев(ВРег(СокрЛП(Отчество)), 1));
КонецПроцедуры
```

> [!note]
> В модуле объекта к реквизитам справочника обращаются **напрямую**
> — просто пишем имя реквизита без дополнительных путей:
> ```bsl
> Фамилия        // реквизит «Фамилия»
> Имя            // реквизит «Имя»
> Наименование   // стандартный реквизит (синоним «ФИО»)
> ```

> [!warning]
> Если отчество не заполнено — в строке появится лишняя точка.
> Для корректной обработки необходимо добавить проверку
> на заполненность отчества.
