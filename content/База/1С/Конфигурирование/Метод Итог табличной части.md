## Метод .Итог() табличной части

Метод суммирует значения всех строк в указанной колонке табличной части.

### Синтаксис

```bsl
ТабличнаяЧасть.Итог("ИмяКолонки")
```

| Элемент | Описание |
|---|---|
| `ТабличнаяЧасть` | Имя табличной части объекта |
| `"ИмяКолонки"` | Имя колонки в виде строки (в двойных кавычках) или её индекс |

### Примеры

```bsl
// Итог по одной табличной части
Товары.Итог("Сумма")

// Итог по двум табличным частям
Товары.Итог("Сумма") + Услуги.Итог("Сумма")

// Итог по количеству
Товары.Итог("Количество")
```

### Практическое применение: реквизит «Итоговая сумма»

Расчёт итоговой суммы документа при записи:

```bsl
Процедура ПередЗаписью(Отказ, РежимЗаписи, РежимПроведения)
    ИтоговаяСумма = Товары.Итог("Сумма") + Услуги.Итог("Сумма");
КонецПроцедуры
```

> [!tip]
> Описание метода можно посмотреть в **Синтакс-помощнике**: Табличная часть → `.Итог()`.
