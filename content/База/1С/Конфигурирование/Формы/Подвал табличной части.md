## Подвал табличной части

**Подвал** — серая полоска под табличной частью, где автоматически отображается итог по выбранной колонке.

> [!info]
> Подвал работает **только в табличных частях**. В динамических списках (например, в списке документов) реализовать его таким способом невозможно.

### Шаг 1: Включить подвал для табличной части

1. В области элементов формы — двойной клик по табличной части (например, **«Товары»**)
2. Перейти на вкладку **«Использование»**
3. Поставить флажок **«Подвал»**
4. В окне предпросмотра снизу появится серая полоска

### Шаг 2: Настроить колонку для отображения в подвале

1. Раскрыть табличную часть в области элементов формы
2. Двойной клик по нужной колонке (например, **«ТоварыСумма»**)
3. Перейти на вкладку **«Основное»**
4. Прокрутить вниз — настроить два свойства:

| Свойство | Значение |
|---|---|
| **Текст подвала** | `Итого:` |
| **Путь к данным подвала** | Выбрать через «...» → Объект → Товары → **Итог Сумма** |

5. Нажать **ОК**

Путь к данным подвала будет выглядеть так: `Объект.Товары.ИтогСумма`

### Пример: подвал для двух табличных частей

**Табличная часть «Товары»:**
- Флажок «Подвал» ✅
- Колонка «ТоварыСумма»: путь `Объект.Товары.ИтогСумма`, текст `Итого:`

**Табличная часть «Услуги»:**
- Флажок «Подвал» ✅
- Колонка «УслугиСумма»: путь `Объект.Услуги.ИтогСумма`, текст `Итого:`

### Настройка ширины колонки

> [!tip]
> Если содержимое не помещается и появляется горизонтальный скролл:
> - В **пользовательском режиме**: кнопка **«Ещё»** → **«Изменить форму»** → задать ширину
> - В **конфигураторе**: выделить элемент → вкладка **«Расположение»** → свойство **«Ширина»** (0 = автоматически)
