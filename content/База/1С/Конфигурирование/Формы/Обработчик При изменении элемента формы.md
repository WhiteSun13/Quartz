## Обработчик ПриИзменении элемента формы

Обработчик события, который срабатывает, когда пользователь **изменяет значение** поля на форме.

### Как создать обработчик

1. Выделить элемент формы (поле Номер / Дата / Покупатель / и т.д.)
2. Правая кнопка мыши → **«События»** → **«При изменении»**
3. Нажать лупу → **«Создать на клиенте»**
4. Написать код обработчика

### Сигнатура

```bsl
&НаКлиенте
Процедура ИмяПоляПриИзменении(Элемент)
    // Код выполняется при изменении значения поля
КонецПроцедуры
```

### Примеры

```bsl
&НаКлиенте
Процедура НомерПриИзменении(Элемент)
    Сообщить("Изменено поле: Номер");
КонецПроцедуры

&НаКлиенте
Процедура ДатаПриИзменении(Элемент)
    Сообщить("Изменено поле: Дата");
КонецПроцедуры

&НаКлиенте
Процедура ПокупательПриИзменении(Элемент)
    Сообщить("Изменено поле: Покупатель");
КонецПроцедуры

&НаКлиенте
Процедура ДоговорПриИзменении(Элемент)
    Сообщить("Изменено поле: Договор");
КонецПроцедуры
```

### Функция Сообщить()

Выводит текстовое сообщение в панель сообщений внизу формы.

```bsl
Сообщить("Текст сообщения");
Сообщить("Изменено поле: Номер");
```

> [!note]
> Обработчик создаётся **на клиенте** (`&НаКлиенте`) — это означает, что код выполняется на стороне пользователя, без обращения к серверу.
