## Обработчик ПередЗаписью документа

Обработчик события модуля объекта, который выполняется **перед сохранением документа**.

### Открытие обработчика

1. Вкладка **«Прочие»** документа
2. Кнопка **«Модуль объекта»**
3. Нажать **лупу** → выбрать **«Перед записью»**

### Сигнатура

```bsl
Процедура ПередЗаписью(Отказ, РежимЗаписи, РежимПроведения)
    // Код выполняется до сохранения документа
КонецПроцедуры
```

---

## Параметр «Отказ»

Если установить в `Истина` — **вся операция аннулируется**: данные не сохраняются, документ не записывается.

```bsl
Процедура ПередЗаписью(Отказ, РежимЗаписи, РежимПроведения)
    ИтоговаяСумма = Товары.Итог("Сумма") + Услуги.Итог("Сумма");

    // Запрет записи если сумма превышает лимит
    Если ИтоговаяСумма > 10000 Тогда
        Отказ = Истина;
    КонецЕсли;
КонецПроцедуры
```

> [!note]
> По умолчанию параметр `Отказ` **содержит `Ложь`** — явно писать `Отказ = Ложь` не нужно.

> [!tip]
> **Практическое применение:** проверка отрицательных остатков на складе. Если товара не хватает — `Отказ = Истина`, документ не проводится, данные в регистр не попадают.

---

## Параметр «РежимЗаписи»

Определяет, что именно происходит при записи документа.

```bsl
РежимЗаписиДокумента.Запись           // Только сохраняет данные документа
РежимЗаписиДокумента.Проведение       // Сохраняет + переносит данные в регистры
РежимЗаписиДокумента.ОтменаПроведения // Убирает галочку проведения + удаляет данные из регистров
```

---

## Параметр «РежимПроведения»

Определяет режим проведения документа.

```bsl
РежимПроведенияДокумента.Оперативный    // Реальное время, будущей датой записать нельзя
РежимПроведенияДокумента.Неоперативный  // Можно записывать прошлой и будущей датой
```

| Режим | Описание |
|---|---|
| **Оперативный** | По умолчанию. Проверяет остатки «здесь и сейчас». Нельзя установить будущую дату |
| **Неоперативный** | Для ввода «задним числом». Дата может быть любой |

> [!note]
> Настройки проведения задаются в свойствах документа на вкладке **«Движение»**: «Проведение разрешено», «Оперативное проведение разрешено».
