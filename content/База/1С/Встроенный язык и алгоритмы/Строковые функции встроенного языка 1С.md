# Строковые функции встроенного языка 1С

## СтрШаблон

Формирует строку по шаблону, подставляя значения на метки `%1`, `%2`, … `%N`.

**Синтаксис:**
```bsl
СтрШаблон(Шаблон, Значение1, Значение2, ...)
```

**Пример:**
```bsl
// Результат: "Иванов И.И."
СтрШаблон("%1 %2.%3.", "Иванов", "И", "И")
```

> [!note]
> Количество меток (`%1`, `%2`, …) должно совпадать с количеством
> переданных значений.

## СокрЛП

Удаляет незначимые пробелы слева и справа от строки.

```bsl
СокрЛП("  Иванов  ")   // → "Иванов"
```

Аналоги:
- `СокрЛ` — только слева
- `СокрП` — только справа

## Лев / Прав

Возвращает указанное количество символов с левого или правого края строки.

```bsl
Лев("Иван", 1)    // → "И"
Прав("Иван", 2)   // → "ан"
```

## ВРег / НРег / ТРег

Изменяют регистр всей строки.

| Функция | Результат |
|---|---|
| `ВРег("иван")` | `"ИВАН"` — верхний регистр |
| `НРег("ИВАН")` | `"иван"` — нижний регистр |
| `ТРег("иван иванов")` | `"Иван Иванов"` — каждое слово с заглавной |

## Практический пример: формирование «Фамилия И.О.»

Задача — из трёх реквизитов сформировать строку вида `Иванов И.И.`:

```bsl
Наименование = СтрШаблон("%1 %2.%3.",
    СокрЛП(Фамилия),
    Лев(ВРег(СокрЛП(Имя)), 1),
    Лев(ВРег(СокрЛП(Отчество)), 1));
```

**Разбор шагов:**

| Шаг | Выражение | Что делает |
|---|---|---|
| 1 | `СокрЛП(Фамилия)` | Убирает лишние пробелы у фамилии |
| 2 | `СокрЛП(Имя)` | Убирает лишние пробелы у имени |
| 3 | `ВРег(...)` | Приводит к верхнему регистру (защита от строчного ввода) |
| 4 | `Лев(..., 1)` | Берёт первую букву |
| 5 | `СтрШаблон` | Собирает итоговую строку по шаблону |

> [!tip]
> Перетащите имя реквизита из дерева конфигурации в модуль
> зажав левую кнопку мыши — платформа автоматически
> подставит корректное имя без опечаток.
