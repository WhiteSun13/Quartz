# Структурирование кода: Области

## Зачем нужны области

- Группируют код в логические блоки — как «папки»
- Позволяют **сворачивать** разделы кода
- Обязательны для соответствия **стандартам разработки 1С**

## Создание области

```bsl
#Область ОбработчикиСобытийЭлементовФормы

// ... обработчики событий ...

#КонецОбласти

#Область СлужебныеПроцедурыИФункции

// ... вспомогательные процедуры ...

#КонецОбласти
```

> [!warning]
> В названии области **не должно быть пробелов**!
> Используйте `ОбработчикиСобытийЭлементовФормы`
> вместо `Обработчики событий элементов формы`.

## Быстрое создание области

1. Написать `#Область` (или аббревиатуру `obl`)
2. Нажать **Ctrl+Q** — откроется помощник

## Если области не сворачиваются

1. Меню **Сервис** → **Параметры**
2. Вкладка **Модули** → **Группировка**
3. Поставить галочку **«Области сворачивать»**
4. Нажать **Применить**
