# Работа с датами в 1С

## Основные функции

| Функция | Описание |
|---|---|
| `ТекущаяДата()` | Возвращает текущую дату и время |
| `Год(Дата)` | Возвращает год из значения типа Дата |
| `Месяц(Дата)` | Возвращает месяц (1–12) |
| `День(Дата)` | Возвращает день месяца |

## Пустая дата

Пустая (нулевая) дата в 1С — `01.01.0001`.
В коде записывается как литерал:

```bsl
'00010101'
```

Проверка на незаполненность:
```bsl
Если Данные.ДатаРождения > '00010101' Тогда
    // дата заполнена
КонецЕсли;
```

## Арифметика дат

Разность двух дат возвращает количество **секунд** между ними:

```bsl
Секунд = ТекущаяДата() - ДатаРождения;
```

К дате можно прибавить количество секунд:

```bsl
НоваяДата = '00010101' + КоличествоСекунд;
```

| Период | Секунд |
|---|---|
| 1 минута | 60 |
| 1 час | 3 600 |
| 1 день | 86 400 |
| 1 год (365 дней) | 31 536 000 |

## Алгоритм расчёта возраста в годах

**Идея:** перевести разность дат в секунды, прибавить к пустой дате,
извлечь год и скорректировать результат.

```bsl
// Шаг 1: Разница в секундах
ВозрастВСекундах = ТекущаяДата() - Данные.ДатаРождения;

// Шаг 2: Прибавляем секунды к пустой дате
Возраст1 = '00010101' + ВозрастВСекундах;

// Шаг 3: Убираем лишний день
// (пустая дата сама содержит 1 день = 86 400 секунд)
Возраст2 = Возраст1 - 86400;

// Шаг 4: Берём только год
Возраст3 = Год(Возраст2);

// Шаг 5: Корректируем на 1
// (иначе возраст будет на 1 год больше реального)
Возраст3 = Возраст3 - 1;
```

> [!warning]
> Без вычитания `1` на шаге 5 возраст будет на **один год больше**
> реального. Например, в день рождения программа покажет `32`
> вместо `31`.
