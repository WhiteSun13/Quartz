# Программный код расчёта суммы

## Обращение к реквизитам табличной части

> [!important]
> Путь к данным: от **большего** к **меньшему**:
> ```
> Элементы → Товары (табличная часть) → ТекущиеДанные → Конкретная колонка
> ```

## Код обработчика «При изменении Количества»

```bsl
&НаКлиенте
Процедура ТоварыКоличествоПриИзменении(Элемент)
    РасчётСуммы();
КонецПроцедуры
```

## Код обработчика «При изменении Цены»

```bsl
&НаКлиенте
Процедура ТоварыЦенаПриИзменении(Элемент)
    РасчётСуммы();
КонецПроцедуры
```

## Вынесенная процедура расчёта суммы

```bsl
&НаКлиенте
Процедура РасчётСуммы()
    СтрокаРасчёта = Элементы.Товары.ТекущиеДанные;
    СтрокаРасчёта.Сумма = СтрокаРасчёта.Цена * СтрокаРасчёта.Количество;
КонецПроцедуры
```

## Почему через `Элементы`, а не через `Объект.Товары`?

> [!note]
> - `Объект.Товары` — работает с **уже записанными** в базу данными
> - `Элементы.Товары.ТекущиеДанные` — работает с **текущей,
>   ещё не записанной** строкой, которую пользователь вводит
>   прямо сейчас

> [!warning]
> **`ТекущиеДанные`** недоступны на сервере! Процедуры,
> использующие `ТекущиеДанные`, **обязательно** должны иметь
> директиву `&НаКлиенте`.
