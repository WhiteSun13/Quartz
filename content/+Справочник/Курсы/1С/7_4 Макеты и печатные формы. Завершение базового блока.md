## 🖨 Что такое макеты и печатные формы

> [!important]
> **Макет** — объект конфигурации, предназначенный для хранения различных форм представления данных. По сути — **шаблон документа**, в некоторые ячейки которого подставляются данные из базы.

**Что может содержать макет:**
- 📋 Табличный документ
- 📝 Текстовый документ
- 💾 Двоичные данные
- 🌐 HTML-документ
- И другие форматы

**Принцип работы:**
```
Шаблон (макет) + Данные из базы → Печатная форма
```

## 🛠 Создание печатной формы для «Расходной накладной»

### Шаг 1. Открытие конструктора печати

1. Двойной клик по документу **«Расходная накладная»**
2. Перейти на вкладку **«Макеты»**
3. Нажать кнопку **«Конструкторы»** (внизу)
4. Выбрать единственный пункт: **«Конструктор печати»**

> [!tip]
> 1С автоматизирует всё, что можно — конструктор печати генерирует шаблон и весь необходимый код автоматически.

---

### Шаг 2. Настройка в конструкторе (пошагово)

**Имя команды:** оставить **«Печать»** → нажать **«Далее»**

---

#### 📌 Страница 1 — Реквизиты шапки

> Шапка — информация, располагающаяся **над таблицей**

Выбрать:
- ☑ **Номер** (номер документа)
- ☑ **Дата**
- ☑ **Покупатель**
- ☐ Договор (не выбираем)

→ Нажать **«Далее»**

---

#### 📌 Страница 2 — Реквизиты табличной части «Товары»

Выбрать (все, кроме номера строки):
- ☑ **Товар**
- ☑ **Количество**
- ☑ **Цена**
- ☑ **Сумма**

> [!note]
> Порядок выбранных реквизитов определяет порядок колонок в печатной форме (слева направо).

→ Нажать **«Далее»**

---

#### 📌 Страница 3 — Реквизиты табличной части «Услуги»

Выбрать:
- ☑ **Услуга**
- ☑ **Сумма**
- ☐ Номер строки (не выбираем)

→ Нажать **«Далее»**

---

#### 📌 Страница 4 — Реквизиты подвала

> Подвал — информация, располагающаяся **под табличными частями**

Выбрать:
- ☑ **ИтоговаяСумма**

→ Нажать **«Далее»**

---

#### 📌 Страница 5 — Группа команды (последняя)

| Настройка | Значение |
|---|---|
| **Группа** | Командная панель формы.Важная (оставить по умолчанию) |
| **Печать без предварительного просмотра** | Документ сразу отправляется на принтер без показа |
| **Таблица «Только просмотр»** | Запрет изменений в сформированном документе |
| **Защита таблицы** | Аналогично «Только просмотр» |

→ Нажать **«Ок»**

---

### ✅ Результат работы конструктора

> [!success]
> После нажатия «Ок» автоматически создаётся:
> 1. **Макет «Печать»** — шаблон документа (в ветке «Макеты» расходной накладной)
> 2. **Команда «Печать»** — кнопка для вызова печатной формы
> 3. **Код в модуле менеджера** — процедура «Печать» с алгоритмом формирования документа

**Структура созданного макета:**

```
Заголовок
└── Шапка (Номер, Дата, Покупатель)
    └── Товары (шапка таблицы)
        └── Строки товаров (Товар, Количество, Цена, Сумма)
    └── Услуги (шапка таблицы)
        └── Строки услуг (Услуга, Сумма)
Подвал (ИтоговаяСумма)
```

## 💻 Структура сгенерированного кода

### Команда (модуль команды)

```
// Краткий код вызова печатной формы
// Открывает табличный документ для просмотра
```

### Процедура «Печать» (модуль менеджера)

```
// Более объёмный алгоритм:

// 1. Получение макета
Макет = ПолучитьМакет("Печать");

// 2. Создание запроса для получения данных из базы
Запрос = Новый Запрос;
// ... текст запроса ...

// 3. Обработка результата:
// - ПолучитьОбласть("Заголовок")
// - ПолучитьОбласть("Шапка")
// - ПолучитьОбласть("ТоварыШапка")
// - Цикл по строкам → подстановка данных в ячейки
// - ПолучитьОбласть("УслугиШапка")
// - Цикл по строкам услуг
// - ПолучитьОбласть("Подвал")
```

> [!note]
> Весь этот код можно было написать вручную. Конструктор печати генерирует его автоматически, экономя время разработчика.

## ⚙️ Дополнительные настройки печатной формы

### 📄 Ориентация страницы

В открытой печатной форме:
1. Нажать **«Ещё»** (кнопка сверху)
2. Выбрать **«Параметры страницы»**
3. Найти **«Ориентация»**:
   - **Портрет** — вертикальная (по умолчанию)
   - **Ландшафт** — горизонтальная (альбомная)

> [!warning]
> При создании макета важно следить за **шириной документа** — данные должны помещаться на листе А4. Если колонок много, может понадобиться альбомная ориентация или уменьшение ширины столбцов.

### 🖨 Отправка на печать

В открытой печатной форме:
1. Нажать кнопку **«Печать»**
2. Выбрать принтер
3. Нажать **«Печать»**

## ❓ Ответы на вопросы по макетам

> [!note]
> **«Как вывести итог под таблицей товаров (а не только в подвале всего документа)?»**
> Стандартными средствами конструктора — нельзя. Варианты:
> 1. Вручную отредактировать макет — перенести область «Подвал» в нужное место
> 2. Создать отдельный реквизит, хранящий итог по конкретной табличной части, и вывести его в нужной области макета

> [!note]
> **«Как запустить конструктор печати, не заходя внутрь документа?»**
> ПКМ по документу в дереве конфигурации → **«Конструкторы»** → **«Конструктор печати»**

> [!note]
> **«Можно ли загрузить макет из Excel?»**
> Существуют специальные обработки для переноса данных из Excel в 1С. Решения легко находятся в интернете по запросу «перенос из Excel в 1С макет».

> [!note]
> **«Как команду «Печать» поместить в нужную группу, если она не отображается?»**
> ПКМ по команде → **«Свойства»** → в поле «Группа» выбрать **«КоманднаяПанельФормы.Важная»**

> [!note]
> **«Существуют ли готовые шаблоны макетов (устанавливаемые отдельно)?»**
> Возможные варианты:
> - Файлы расширений **`.cfe`** — содержат наборы макетов, подключаемых к конфигурации
> - Внешние обработки с макетами (файлы `.epf`)
> - XML-файлы с описанием шаблонов

> [!note]
> **«Как вывести команду печати для макета с типом «Схема компоновки данных»?»**
> Это отдельная область применения — печатные формы и СКД используются в разных контекстах и, как правило, не пересекаются напрямую.
